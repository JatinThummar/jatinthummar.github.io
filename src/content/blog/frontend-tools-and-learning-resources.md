Beyond individual blogs, these are the communities, curated collections, and learning platforms that I keep bookmarked. Whether you're looking for tutorials, videos, templates, or just a place to ask questions — this list has you covered.

## React

- **[Awesome React](https://awesomereact.com/)** — The best React videos and playlists on the web, curated and organized by topic.
- **[React Resources](https://reactresources.com/)** — Curated collection of React articles, tutorials, tools, and libraries.
- **[Reactiflux](https://www.reactiflux.com/)** — One of the largest React communities. Great place to ask questions and connect with other developers.
- **[Awesome React Components](https://github.com/brillout/awesome-react-components#tooltip)** — Curated list of React components — tooltips, modals, forms, and more.
- **[Collection of React Hooks](https://nikgraf.github.io/react-hooks/)** — Comprehensive collection of community React hooks.

## Next.js

- **[Next Radar](https://nextradar.dev/content/nextjs)** — Top resources, templates, courses, and jobs for Next.js and React.
- **[Awesome Next.js](https://awesomenextjs.com/)** — Curated list of Next.js resources — boilerplates, articles, extensions, and more.
- **[Vercel Community](https://community.vercel.com/)** — Official Vercel community forum. Good for Next.js help and deployment questions.

## Remix

- **[Remix Guide](https://remix.guide/)** — Curated resources and community links for the Remix framework.

## JavaScript & Web Fundamentals

- **[The Modern JavaScript Tutorial](https://javascript.info/)** — Comprehensive, beginner-to-advanced JavaScript tutorial. One of the best free resources.
- **[JavaScript to Know for React](https://kentcdodds.com/blog/javascript-to-know-for-react)** — Kent C. Dodds breaks down the JS concepts you actually need for React.
- **[Epic React Articles](https://epicreact.dev/articles)** — Articles from Kent C. Dodds' Epic React workshop series.
- **[web.dev — CSS Box Model](https://web.dev/learn/css/box-model/)** — Google's learn CSS series. Clear explanations of core CSS concepts.
- **[Complete Intro to React v7](https://btholt.github.io/complete-intro-to-react-v7)** — Brian Holt's free, comprehensive React course from Frontend Masters.

## React Native

- **[React Native Shadow Generator](https://ethercreative.github.io/react-native-shadow-generator/)** — Generate cross-platform shadow styles for React Native.
- **[React SVGR Playground](https://react-svgr.com/playground/?native=true&ref=true&typescript=true)** — Convert SVGs to React Native components instantly.

## Frontend Architecture & Patterns

- **[Frontend at Scale](https://frontendatscale.com/)** — Level up your frontend architecture skills. Focused on scaling frontend systems and teams.
- **[Refactoring Guru](https://refactoring.guru/)** — Design patterns and refactoring techniques explained with clear visuals. Not frontend-specific, but essential knowledge for writing clean, maintainable code.
- **[Jamstack Generators](https://jamstack.org/generators/)** — Compare every static site generator. Useful for choosing your next framework.

## General Learning & Tools

- **[Happy Coding](https://happycoding.io/tutorials/how-to/learn-coding)** — Friendly tutorials on how to start learning to code.
- **[Free Programming Courses](https://github.com/EbookFoundation/free-programming-books/blob/main/courses/free-courses-en.md#nextjs)** — Massive curated list of free programming courses, including Next.js and React.
- **[Product Hunt](https://www.producthunt.com/)** — Discover new developer tools and products every day.
- **[Toolfolio](https://toolfolio.io/)** — All the tools you need in one place. Searchable directory of developer tools.
- **[Prompt Engineering Guide](https://www.promptingguide.ai/introduction/settings)** — Learn how to effectively prompt AI models. Increasingly essential for dev workflows.
