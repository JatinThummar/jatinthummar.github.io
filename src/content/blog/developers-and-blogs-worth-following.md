These are the developers and bloggers I've learned from over the years. Whether you're into React Native, Next.js, or frontend engineering in general — there's something here for everyone.

## React & Frontend Fundamentals

- **[Josh W Comeau](https://www.joshwcomeau.com/)** — CSS, React, and interactive tutorials. One of the best at making complex concepts click.
- **[Dan Abramov](https://overreacted.io/)** — React core team. Deep, thoughtful writing about React internals and programming in general.
- **[The Joy of React](https://www.joyofreact.com/)** — Learn React the right way. Personal favorite, recommended by Delba at Vercel.
- **[Mark Erikson](https://blog.isquaredsoftware.com/)** — Redux maintainer. Incredibly detailed posts on React, Redux, and state management.
- **[Kent C. Dodds](https://www.epicai.pro/posts)** — Testing, React patterns, and now AI engineering. Prolific educator.
- **[Robin Wieruch](https://www.robinwieruch.de/)** — Practical React and JavaScript tutorials. Great for learning by building.
- **[Philip Roberts](http://latentflip.com/)** — Famous for the JS event loop explainer video. Writes about JavaScript and web fundamentals.

## Expo & React Native

- **[Evan Bacon](https://evanbacon.dev/)** — Expo core team. Pushing the boundaries of what's possible with Expo and universal React.
- **[Rafael Mendiola](https://raf.dev/)** — React Native animations and mobile dev.
- **[Fernando Rojo](https://fernandorojo.co/conf)** — Built a startup from zero to $10M with React Native + Next.js. Real-world insights.
- **[Reactiive](https://reactiive.io/)** — React Native tutorials and component breakdowns.
- **[kelset (Lorenzo Sciandra)](https://kelset.dev/)** — React Native core contributor. Writes about the ecosystem and open source.
- **[Christoph Nakazawa](https://cpojer.net/)** — Created Jest, contributed to React Native. Open source veteran and game developer.

## Next.js & Vercel

- **[Lee Robinson](https://leerob.com/vercel)** — VP of Product at Vercel. 5 years of Next.js lessons and web development insights.
- **[Guillermo Rauch](https://rauchg.com/)** — CEO of Vercel, creator of Next.js and Socket.IO. Short, sharp posts on the future of the web.
- **[Aurora Scharff](https://aurorascharff.no/)** — Next.js, React Server Components, and modern web patterns.
- **[Hamed Bahram](https://www.hamedbahram.io/)** — Next.js tutorials and full-stack web development.
- **[Jimmy Lai](https://jimmyl.ai/contact)** — Next.js core team at Vercel.
- **[Wyatt Johnson](https://wyattjoh.ca/)** — Next.js core team at Vercel.

## JavaScript, CSS & Web Platform

- **[Ryan Florence](https://medium.com/@ryanflorence)** — Co-creator of React Router and Remix. Deep understanding of web fundamentals.
- **[Cory House](https://medium.com/@housecor)** — React, JavaScript, and developer experience. Clean, actionable advice.
- **[Swizec Teller](https://swizec.com/)** — React, data visualization, and senior engineer career advice.
- **[Jason Miller](https://jasonformat.com/)** — Creator of Preact. Performance-focused web engineering.
- **[Sam Selikoff](https://samselikoff.com/)** — Tailwind CSS, animations, and frontend craft.
- **[Rodrigo Pombo](https://pomb.us/)** — Code animations, Code Hike. Beautiful visual storytelling with code.
- **[James Long](https://archive.jlongster.com/)** — Creator of Prettier. Writes about programming languages, tooling, and local-first software.
- **[Ryan Carniato](https://www.youtube.com/@ryansolid)** — Creator of SolidJS. A gold mine of knowledge about reactivity and framework design.

## Web Performance & Architecture

- **[Addy Osmani](https://addyosmani.com/)** — Chrome team at Google. Web performance, design patterns, and engineering leadership.
- **[Houssein Djirdeh](https://houssein.me/)** — Web performance and developer tooling at Google.
- **[Destroy All Software](https://www.destroyallsoftware.com/screencasts)** — Gary Bernhardt's screencasts. Deep dives into software concepts in short, focused sessions.
- **[SurviveJS](https://survivejs.com/)** — Books and tutorials on webpack, React, and the JavaScript ecosystem.

## Framework & Ecosystem Contributors

- **[Ben Holmes](https://bholmes.dev/)** — Astro core team. Web framework design and content-driven development.
- **[Anthony Fu](https://antfu.me/)** — Vue, Vite, Nuxt core team. Incredibly prolific open source contributor.
- **[Drew Powers](https://pow.rs/)** — Astro contributor, Open Props. Web standards and modern CSS.

## Engineering Blogs & Career

- **[The Pragmatic Engineer](https://blog.pragmaticengineer.com/)** — Gergely Orosz. Deep coverage of tech industry, engineering culture, and career growth.
- **[Cassidy Williams](https://cassidoo.co/)** — Developer experience, career advice, and web development.
- **[Michele Riva](https://www.micheleriva.it/)** — Author of *Real-World Next.js*. Full-stack engineering and open source.

---

These are people I keep going back to. Their writing, talks, and open source work have shaped how I think about building software. If you're a frontend or mobile developer, following even a few of them will level up your craft.
